# 佇列 (Queue)

- 在資料結構與演算法中， **佇列 (Queue)** 是一種線性資料結構，遵循 **先進先出 (FIFO, First-In-First-Out)** 的原則。
- 在佇列中，第一個加入的元素將會是第一個被移除的，這就像排隊一樣，先到的人會先被服務。

### 佇列的特點
- **FIFO順序**：元素從尾端（後端）加入，並從前端移除。
- **基本操作**：
  - **Enqueue**：將元素加入佇列的尾端。
  - **Dequeue**：從佇列的前端移除元素。
  - **Peek/Front**：查看佇列的前端元素，但不移除。
  - **IsEmpty**：檢查佇列是否為空。
  - **Size**：返回佇列中元素的數量。

### 佇列的應用
佇列廣泛應用於以下場合：
1. **任務排程**：例如在操作系統中管理進程。
2. **資料緩衝**：處理數據流，例如I/O緩衝區。
3. **廣度優先搜尋 (BFS)**：在圖形遍歷中應用。
4. **列印緩衝**：管理列印文件的順序。
